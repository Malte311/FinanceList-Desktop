Benutzerprofil umbenennen
Rename User Profile
Bitte geben Sie einen neuen Namen für das Benutzerprofil "%%USER%%" ein. Der Name darf nicht leer sein, maximal %%MAXLEN%% Zeichen lang sein und darf ausschließlich Buchstaben und Zahlen enthalten (Leerzeichen sind nicht erlaubt). Please type in the new name for the user profile "%%USER%%". The name must not be empty, it must not be longer than %%MAXLEN%% characters and it must only contain letters and digits (no spaces allowed).

Delete User Profile

Um das Benutzerprofil "%%USER%%" zu löschen, müssen Sie unten die Checkbox aktivieren und den Text "%%USER%%" eingeben. Are you sure you want to delete the user profile "%%USER%%"? All of its data will be lost and this action cannot be undone. In order to confirm, activate the checkbox below and type in the text "%%USER%%".
