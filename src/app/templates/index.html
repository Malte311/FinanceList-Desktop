Die "Überblick" Seite

The overview page

Diese Seite soll einen groben Überblick über die aktuelle Lage bieten. Dazu werden folgende Ansichten angeboten:

This page gives you a rough overview about the current situation. The following views are offered:

  • Monatlicher Überschuss

    • Diese Ansicht zeigt die im aktuellen Monat vorhandenen Überschüsse (grün) bzw. Verluste (rot) an.

  • Kürzliche Ausgaben

    • Hier werden kürzlich getätigte Ausgaben aufgelistet.

  • Gesamtausgaben

    • Dieses Diagramm visualisiert alle bisher getätigten Ausgaben.

  • Kürzliche Einnahmen

    • Hier werden kürzlich generierte Einnahmen aufgelistet.

  • Gesamteinnahmen

    • Dieses Diagramm visualisiert alle bisher generierten Einnahmen.

  • Monthly surplus

    • This view displays the surplus (green) respectively loss (red) in the current month.

  • Recent spendings

    • This section lists all recent spendings.

  • All time spendings

    • This diagram visualizes the total amount of spendings done so far.

  • Recent earnings

    • This section lists all recent earnings.

  • All time earnings

    • This diagram visualizes the total amount of earnings generated so far.

Die "Bilanzen" Seite

The balances page

Auf dieser Seite befinden sich genauere Informationen zu Kontostand und zu allen Einnahmen und Ausgaben. Dazu gibt es folgende Ansichten:

This page contains detailed information about current balances and all earnings and spendings. Available views are:

  • Kontenübersicht

    • Hier werden alle verfügbaren Konten mit ihrer jeweiligen Bilanz angezeigt. Zudem besteht die Möglichkeit, Konten hinzuzufügen, zu löschen oder umzubenennen. Über die Schaltfläche "Eintrag hinzufügen" können neue Einnahmen oder Ausgaben hinzugeügt werden. Für Einnahmen gibt es die Option, den Betrag auf verschiedene Konten zu verteilen, wobei die Verteilung nach Belieben angepasst werden kann (dies geschieht unter "Automatische Verteilung"). Weiter kann man Transaktionen automatisieren, sodass sie in einem gewählten Intervall immer wieder von alleine wiederholt werden. Mittels "Transfer" ist es möglich, einen gewählten Betrag zwischen bestehenden Konten zu transferieren.

  • Detailübersicht

    • In diesem Abschnitt können detaillierte Informationen zu allen Ausgaben und Einnahmen abgerufen werden. Die Daten werden entweder in Form eines Diagramms oder in Form einer Tabelle visualisiert. Bei der Tabelle besteht die Option, einzelne Einträge wieder zu löschen, falls ein Eintrag nicht korrekt hinzugefügt wurde. Zudem kann die Tabelle nach einer beliebigen Spalte sowohl aufsteigend als auch absteigend sortiert werden. Dazu klickt man einfach auf die Spaltenüberschrift der jeweiligen Spalte. Die Daten können außerdem beliebig gefiltert werden, wobei es keine Rolle spielt, ob sie in Form eines Diagramms oder in Form einer Tabelle dargestellt werden.

  • Wiederkehrende Transaktionen

    • In dieser Ansicht werden alle wiederkehrenden Transaktionen aufgelistet, um den Überblick darüber nicht zu verlieren. Selbstverständlich können diese wiederkehrenden Transaktionen auch bearbeitet oder gelöscht werden.

  • Budget overview

    • This section lists all available budgets with their current balance. You can add new budgets, rename budgets or delete budgets here as well. You can add new earnings or spendings via "Add transaction". Earnings can be partitioned and distributed to multiple budgets as you like. You can choose the allocation rate by selecting "Automated allocation". Any transaction can be automated such that it gets executed repeatedly in an interval you can choose. You can also transfer a selected sum from a budget to another budget by clicking on the "Transfer" button.

  • Detailed overview

    • This section provides detailed information for all of your earnings and spendings. The data is visualized either by a diagram or by a table. If you choose to display a table, you can delete entries you added incorrectly. In addition to that, the table can be sorted ascending or descending by any column. In order to do exactly that, one simply clicks on the corresponding column heading. It is possible to filter the data by any parameter at any time.

  • Recurring transactions

    • This view lists all of the current recurring transactions such that you do not lose track of them. Of course, you can edit them or delete them here if you want to.

Die "Einstellungen" Seite

The settings page

Auf dieser Seite können Sie Einstellungen vornehmen. Sie können hier auswählen, ob das Programm im Vollbildmodus oder im Fenstermodus laufen soll. Darüber hinaus können der Dateispeicherort für Ihre Daten sowie die Währung und die Diagrammart beliebig gewählt werden.

On this page, one can adjust the settings of this application. You can choose whether this application should be run in fullscreen or in window-mode. Additionally, you can change the path where your data is being saved, the type of diagrams which are used to visualize data and the currency.
